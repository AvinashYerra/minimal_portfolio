Enabling Analytics for your website

Date: 2025-12-07

Whenever you host a website, along with the functionality the analysis that can be done with the visitors is also important

There are many analytics tools available, both paid and free. Google Analytics is one of the most popular ones, but it has some privacy concerns.

Umami is a free and open sourceweb analytics tool that is privacy-focused. It is easy to set up and use.

Steps to enable Umami Analytics for your website

To setup Umami analytics, follow the below steps

  • Setup a postgres instance - I used neondb as this offers a free tier instance and easy to connect
  • Create a repository in your Github and add a file named 'Dockerfile' which has the docker file that is used to initialize Umami
  • Paste the below code in the 'Dockerfile'
    • FROM ghcr.io/umami-software/umami:latest
  • Now host this website using Render (free tier) and add three environment variables for Database connection
    • DATABASE_URL - The connection string for your postgres database
    • APP_SECRET - A random string with size - 32 used for encryption
    • PORT - Set this to 3000
  • Once the build is done and your website is hosted use the default credentials of Umami
    • Username : admin
    • Password : umami
  • Change the password after first login
  • Add a new website in Umami dashboard and click on edit website option
  • Navigate to the script in the edit option
  • Paste this script tag in all the html files of your website in the head
  • Now you can see the analytics data in your Umami dashboard

That's it! You have successfully enabled Umami analytics for your website.

For more details, visit the official Umami website
