Language Learning and DSA Preparation Guide

When doing DSA, the language doesn't really matter. I started with C++, but people prefer Java because it can also be used in web development. So for syntax practice, people prefer writing in Java.

General Use Cases

  • C++: Hardware programming
  • Java: Web development
  • Python: Data Analysis, Machine Learning, etc.

(Use cases can overlap, this is just for a basic idea)

But for college placements, companies don't really care much about the language you know (C++/Python/Java) because most of the time they hire freshers. What matters is how you approach a problem and frame the solution.

When we talk about solving sum, count, etc., that's just for your idea. What matters is how much you've learned and how you approach the solution. Chase quality over quantity.

DSA Study Plan

During the 3rd year internship season or 4th year placement season, if you have some time for coding, start solving the Striver SDE Sheet (~190 Questions):
Link: Striver SDE Sheet

While solving the sheet, make notes of any tricks, logics, or concepts for easier revision later.

If you have enough time, continue with the Striver A2Z Sheet (~450 Questions):
Link: Striver A2Z Sheet

Note: There is no need to take any paid courses. Everything is available for free on the internet.
